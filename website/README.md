# SkillRACE website

This folder contains the public research website for SkillRACE.

## Files

- `index.html` — main project page
- `styles.css` — premium, minimal styles for a clean single-page research landing page
- `script.js` — lightweight reveal + counter animation for protocol highlights

## Run locally

```bash
cd /home/jim/skillrace/website
python3 -m http.server 8080
```

Open `http://localhost:8080/` in a browser.

## Notes

- The page intentionally uses placeholders for final metrics and headline results.
- The content is synchronized to the protocol and designed for clarity with minimal text density.
- Layout uses static block-first visuals (no external graphics, no SVG chart embeds) and CSS-only motion.

## Purpose

- Publicly present the idea and protocol clearly.
- Promote methodology and reproducibility practices for citation.
- Keep headline metrics as explicit placeholders until paid headline campaigns complete.

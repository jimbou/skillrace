Create /workspace/loadcfg.py invoked as `python3 loadcfg.py <path>`. The INI config must have a [server] section with keys host (string) and port (integer). On valid input print 'host=<host> port=<port>'. On a missing section/key or non-integer port, exit non-zero and print an error NAMING the problem (no traceback). float.ini has a float port (not a valid integer).

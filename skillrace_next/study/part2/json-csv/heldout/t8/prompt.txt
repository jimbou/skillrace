Create /workspace/convert.py invoked as `python3 convert.py <in.json> <out.csv>`. It reads a JSON array of flat objects and writes CSV: header = all keys that appear, sorted; one row per object; missing keys become empty cells; fields with commas/quotes/newlines are quoted so the CSV parses back. A field value itself contains a newline; it must be quoted so the CSV still has one row.

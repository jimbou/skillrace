# Build a Python CLI tool

1. Read the task to understand exactly what the tool must do.
2. Implement it as a runnable Python script (e.g. `cli.py`) in `/workspace`, using
   `argparse` for arguments and a `--help` message.
3. The script must run as `python cli.py ...`.
4. **Test it by running it** on a simple example input before finishing.
5. Report how to run it and what it does.

Rules:
- Implement a genuinely working tool — do not leave stubs or `pass` bodies.
- Verify by actually running the script; do not assume it works.

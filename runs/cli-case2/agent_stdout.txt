All tests pass. Here's a summary of what was implemented and verified:

### `/workspace/cli.py` — Disk Usage CLI Tool

| Feature | Details |
|---|---|
| **Positional `path`** | Defaults to current directory; can also be passed as `--path <dir>` |
| **`--format`** | Choices: `human` (e.g. "5 GB") or `bytes` (raw integer). Default: `human` |
| **`--min-space BYTES`** | Checks available free space via `shutil.disk_usage()`; warns on stderr and exits with code 1 if below threshold |

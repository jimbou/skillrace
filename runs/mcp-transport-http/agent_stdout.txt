**Key structural changes:**
1. **New dependency** — `@hono/node-server` was installed (required by the SDK's `StreamableHTTPServerTransport` for Node.js HTTP ↔ Web Standard conversion).
2. **Transport lifecycle** — One long-lived `StreamableHTTPServerTransport` instance is created and connected once, then reused for all incoming HTTP requests.
3. **Session mode** — Configured as stateless (`sessionIdGenerator: undefined`) so no session IDs are managed client-side.
4. **Error handling** — Each request is wrapped in try/catch with proper `500` fallback responses.
